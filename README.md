# DBM-Programming

Welcome and this is the Official Repository for the Discord Bot Maker Programming for the channel { Easy Programming }




To get the commands in. Create a new command or event in DBM, Once that is done go to the List on the left, and right click on the command or event you created and then click "Edit Raw Data" then go to the command or event file you want to add and copy the text in there, and replace the code in the Raw Data with the text you copied, once tha tis done click "save data" and you are all set!
